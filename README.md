# Running the tests locally

1. Clone this repos using `git clone https://github.com/g-s01/TemplateLibrary.git`
2. Install [GoogleTest](https://google.github.io/googletest/)
3. Build the CMake using
   ```
   chmod +x coverage.sh
   ./coverage.sh
   ```
   at the root
